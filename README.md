# WebTravel

Ce projet est une application web de gestion de voyages développée dans le cadre du BUT Informatique à l'IUT d'Orsay.  
Il utilise Django pour le back-end et Bootstrap 5 pour le front-end.

## Installation

1. Cloner le dépôt :
```bash
git clone https://git.iut-orsay.fr/r5a05tp5b/mjadid/webtravel.git
```

2. Créer un environnement virtuel et l'activer :
```bash
python -m venv env
source env/bin/activate  # Sur Windows : env\Scripts\activate
```
